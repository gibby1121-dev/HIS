# HIS

Regenerative soil-biology venture — research, design, and pitch materials.

## Knowledge map

`knowledge-os.html` is a self-contained, interactive graphical map of everything in
this repository — the regenerative soil-biology venture (thesis, evidence with
confidence ratings, product, plan, open questions) and the Sandhills market-snapshot
pipeline (data stages and market signals). Open it in any browser; no install or
network required. Drag to pan, scroll to zoom, click a node to explore its
connections, search, and filter by type.

`systems-architecture.html` is a complementary layered view: it draws both
systems — the soil-biology venture and the market-snapshot pipeline — level by
level (Sources → Ingest → Process → Persist → Serve → Consume), showing each
component's role and responsibility, the data flow down each domain, the
venture's data-flywheel feedback loop, and the cross-domain interactions. Click
any component for its role, responsibility, inputs/outputs, and artifacts.
